def summ(n):
    if n == 0:
        return 1
    return summ (n -1) +  (1/(2*n + 1) *(-1)**n)

def pi(n):
    return 4*summ(n)

- Schreiben Sie unter Verwendung einer while-Schleife eine iterative Python-Funktion, die bei Eingabe von n die Summe der ersten n Folgegliederm also R(n) berechnet und als Ergebnis zurückgibt

def pi(n):
    res = 0
    while  n >= 0:
        res  +=  (-1)**n * 1/(2*n+1)
        n -= 1
    return 4*res
